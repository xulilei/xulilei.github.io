Mybatis之动态sql

何为动态sql？

比如一个查询语句

如果当我们查询 id=1，gender=男l时，中间的小明以及and都需要删除，带来了非常多不方便的地方。

三大元素

if

在where字句之后作为条件判断的语句，其中字段可以防止出现多余的”and”

作为更新语句的条件判断，其中字段可以防止出现多余的”,”

choose

有时候，我们不想使用所有的条件，而只是想从多个条件中选择一个使用。针对这种情况，MyBatis 提供了 choose 元素，它有点像 Java 中的 switch 语句。

foreach

使用场景为对集合进行遍历

批量添加
批量查询
批量删除
批量更新

需要注意的是，批量更新是多条update sql语句，所以需要sql语句写入for each循环内，并且sql语句之间的分割使用“;”号
